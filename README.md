SNAKE-WATER-GUN game
Overview: 
This Python program implements a simple text-based game called "Snake-Water-Gun," where a player chooses between Snake (S), Water (W), or Gun (G), and competes against the computer. The game follows the rules: Snake beats Water, Water beats Gun, and Gun beats Snake.

How to Play:
i)Run the program.
ii)Enter your choice by typing either 'S' for Snake, 'W' for Water, or 'G' for Gun.
iii)The computer will randomly select its choice.
iv)The program will compare your choice with the computer's choice and determine the winner based on the game rules.

Rules:
Snake (S) beats Water (W)
Water (W) beats Gun (G)
Gun (G) beats Snake (S)

Usage:
Clone or download the repository.
Run the Python script.
Follow the on-screen instructions to input your choice.
The program will display both your choice and the computer's choice.
The result of the game (Win, Lose, or Draw) will be shown.

Error Management:
If an invalid choice is entered, the program will display an error message and exit.
